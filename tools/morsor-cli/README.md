# morsor-cli (Python)

Small HTTP client for the Morsor API. Uses **Python 3 standard library only** (no `pip install`).

Run from repo root:

```bash
./scripts/morsor-cli --help
```

Or:

```bash
python3 tools/morsor-cli/morsor_cli.py --help
```

Install into a directory on `PATH`:

```bash
./scripts/install-morsor-cli /usr/local/bin
```
